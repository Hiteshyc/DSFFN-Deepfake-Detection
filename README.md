## 📌 Overview
Deepfake technology has rapidly advanced, posing serious challenges to digital trust and media authenticity.  
This project implements the **Dual-Stream Forgery Fusion Network (DSFFN)**, a deep learning framework that enhances deepfake detection by leveraging both:

- **Spatial domain features (RGB images)**
- **Frequency domain features (phase spectrum)**

The fusion of these complementary representations improves robustness and reduces performance degradation on unseen datasets.

---

## ✨ Key Features
- Dual-stream architecture (Spatial + Frequency)
- EfficientNet-B0 backbone for feature extraction
- Feature-level fusion using concatenation
- Strong cross-dataset generalization capability
- End-to-end training pipeline in PyTorch
- Evaluation using **Accuracy** and **AUC (ROC)**
